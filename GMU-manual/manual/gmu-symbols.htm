Symbols for _CountCompile.gmu.cnt ($(gmu_Gp_COUNT_COMPILE))

The chars logged in this file is used to report the compile & link summary at the end of GMU run -- that is:

[[[END]]]                                        Projects:16,Containers:30+1
Files compiled: C++:107 C:3 ASM:0 linked:15
CVS-action:0+0,SVN-action:11+4
symbol meaning
c compiled a C file
x compiled a C++ file
a compiled an assembler file
l linked a EXE, DLL, or LIB
+ a non-container project is done and something is linked in that project. Currently, an 'l' will be followed by a '+'.
= a non-container project is done and nothing is linked(i.e. up-to-date) in that project
R done a container project, which has GMU sub-projects
r done a container project, which has no GMU sub-projects (gmu_DELIBERATE_NO_SUBPRJ=1)
V done a CVS checkout/export of sub-project makefiles
v done an additional CVS checkout /export within a project (gmu_uv_list_CVS_MODULES)
S done a SVN checkout/export of sub-project makefiles
s done an additional SVN checkout /export within a project (gmu_uv_list_SVN_MODULES)
