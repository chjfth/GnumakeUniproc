
Normally, using GnumakeUniproc, you apply the same set of compiler flags(cflag) 
to all source files in a project. But sometimes, you may want to add special 
cflags for only one/some source files. 

This example shows you how to add special cflag -DINPUT_NUM=7 only to 2.c in the project.

See this page for detail: http://gnumakeuniproc.sourceforge.net/wiki/index.php/How_do_I_set_special_cflags_to_only_some_specific_source_files
