How to use EmEditor syntax file

Introduction

I, creator of GnumakeUniproc, use EmEditor(a great shareware text editor from http://www.emeditor.com/) to edit GMU makefiles daily. EmEditor has great text editing and visualizing features that ease my daily editing work drastically.

With GMU-Makefile syntax highlight, you achieve these benefits:

  • GMU reserved variables are rendered in special color, so that you can easily know whether some gmu-var is spelled correctly.
  • Space and tab are marked with different marker, so that you can easily know whether you type them right.
  • Large chunks of comments are rendered in dark green, so that you can focus on true makefile statement.

Install

Just import GMU-Makefile.reg into your registry(double click and answer yes). Choose GMU-Makefile syntax configuration (Tools -> Configuration -> GMU-Makefile) or from toolbar button.

Then you get

The .reg write registry keys only inside [HKEY_CURRENT_USER\Software\EmSoft\EmEditor v3\Config\GMU-Makefile], so it cannot cause harm to your system.

The provided .reg has been tested with EmEditor v11. It can be used with older versions of EmEditor as early as v6, but visual effects may be degraded.

How to tweak

If you are not satisfied with my predefined color scheme, you can do some tweak. From program menu, Tools -> Properties for Current Configuration, it pops up a configuration dialog that lets you adjust every aspect of "GMU-Makefile", quite many things you can tweak.
